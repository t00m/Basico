From source code

  1. Clone project: git clone https://github.com/t00m/Basico.git

  2. Install it:

    1. Local: python3 setup install --user (recommended)

    2. System: sudo python3 setup install

  3. Execute it:

    1. Local: $HOME/.local/bin/basico

    2. System: /usr/bin/basico

From Pypi

  • Optionally, uninstall previous versions: pip3 uninstall basico -qy

  • Install:

    • Local: pip3 install basico --user

    • System: sudo pip3 install basico
